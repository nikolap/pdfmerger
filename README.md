## PDFMerger

PDFMerger has one function only - to merge several PDF files into one. Simplicity is key.

## Dependencies

* [PyPDF2](https://github.com/mstamy2/PyPDF2)
* [PySide (requires Qt)](http://qt-project.org/wiki/PySide)
